## 🌟 项目简介  
**民法典助手** 是一款专注中国《民法典》的微信小程序，为用户提供：  
✅ **精准法条查询**｜按编/章/条快速定位  
✅ **场景化法律解读**｜婚姻、继承、合同、侵权等高频场景  
✅ **合规风险提示**｜日常行为法律边界预警  
✅ **案例参考库**｜最高人民法院指导案例精要  
✅ **隐私保护**｜所有查询本地处理，无用户数据上传  

> 💡 **设计原则**：专业严谨 × 通俗易懂 × 零法律基础友好  
> 📌 **免责声明**：本工具提供法律知识参考，不构成正式法律意见。重大事项请咨询执业律师（拨打 **12348**）。

---

## 🚀 核心功能  
| 模块 | 说明 |  
|------|------|  
| **智能检索** | 支持关键词/法条号/场景描述多维度查询 |  
| **法条全景** | 完整《民法典》7编1260条 + 司法解释关联 |  
| **情景问答** | “租房押金不退怎么办？”“离婚财产如何分？”等200+高频问题 |  
| **风险自检** | 合同签署、遗嘱订立等场景合规检查清单 |  
| **收藏笔记** | 重点法条标注+个人笔记（本地存储） |

## 🛠 技术栈  
```markdown
- **前端框架**：微信小程序原生 (WXML/WXSS/JS)  
- **核心逻辑**：  
  - `utils/`：法条数据库解析、搜索算法、风险计算引擎  
  - `components/`：法条卡片、情景问答组件、合规检查器  
- **数据源**：  
  - 《中华人民共和国民法典》官方文本（2021实施版）  
  - 最高人民法院指导案例库（脱敏处理）  
- **开发规范**：ESLint + `.gitignore` 排除敏感配置
